# 个人任务管理系统

## 项目简介
基于Vue.js + Node.js的全栈任务管理应用，支持任务分类、日历视图、数据统计等功能。

## 技术栈
- **前端**: Vue.js 3 + Element Plus + Vite
- **后端**: Node.js + Express + MySQL
- **数据库**: MySQL 8.0

## 功能特性
- ✅ 任务CRUD操作
- ✅ 任务分类管理
- ✅ 日历视图
- ✅ 数据统计图表
- ✅ 用户认证

## 目录结构
```
├── backend/          # 后端API
├── frontend/         # 前端界面
└── docs/            # 项目文档
```

## 快速开始
### 后端
```bash
cd backend
npm install
npm run dev
```

### 前端
```bash
cd frontend
npm install
npm run dev
```
